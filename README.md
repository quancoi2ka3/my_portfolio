# 🚀 Personal Portfolio Project

Một dự án portfolio cá nhân tôi làm với sự hỗ trợ của AI và triển khai trên Vercel chỉ cần vài phút thực hiện

- Thiết kế responsive  
- Giao diện lịch sự  
- Được hỗ trợ bởi công nghệ AI  
- Dễ dàng tùy chỉnh  

## 🛠️ Công Nghệ  

- React, Tailwind CSS
- Deployment: Vercel
